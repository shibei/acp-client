## ⚙️ 打包参数说明

GUI程序使用了以下PyInstaller参数：
- `--onefile`: 打包成单个exe文件
- `--windowed`: 无控制台窗口（GUI程序）
- `--name="观测计划调度器"`: 设置程序名称

## 🔧 重新打包

如果需要重新打包，可以使用以下命令：

```bash
# 打包GUI程序
pyinstaller --onefile --windowed --name="观测计划调度器" observation_scheduler_tkinter_gui.py

# 打包基础可视化器
pyinstaller --onefile --name="基础可视化器" observation_scheduler_visualizer.py

# 打包高级可视化器
pyinstaller --onefile --name="高级可视化器" observation_visualizer_advanced.py
```
